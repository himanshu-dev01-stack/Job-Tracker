# Job-Tracker
It is a job tracking app, which is a fullstack project

## Features
- Track job applications
- Manage job listings
- User authentication

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/job-tracker.git
   ```
2. Navigate to the Client and Server directories and install dependencies:
   ```bash
   cd Client
   npm install
   cd ../Server
   npm install
   ```

## Usage
1. Start the server:
   ```bash
   node server.js
   ```
2. Start the client:
   ```bash
   npm run dev
   ```
